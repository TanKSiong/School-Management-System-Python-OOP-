# 🎓 School Management System (Python OOP)

## 📌 Overview

This project is a School Management System developed using Python, designed to manage student records, academic data, class schedules, and staff information.

The system applies Object-Oriented Programming (OOP) principles to organize different entities such as students, teachers, and administrators, while using CSV files for data storage and management.

---

## 🚀 Features

* Student information management
* Teacher and employee records handling
* Class scheduling system
* Academic records tracking
* File-based data storage using CSV
* Modular OOP-based design

---

## 🛠 Tech Stack

* Programming Language: Python
* Paradigm: Object-Oriented Programming (OOP)
* Data Storage: CSV files
* Libraries: Built-in Python modules (e.g., `csv`)

---

## 📂 Project Structure

```id="u9w4i2"
project/
│── main.py                # Entry point of the system
│── school.py              # Core system logic
│── person.py              # Base class
│── student.py             # Student class
│── admin.py               # Admin functionality
```

## 📊 System Workflow

1. System initializes and loads CSV data
2. User interacts through the main program
3. System processes operations (add/update/view records)
4. Data is stored and updated in CSV files
5. Output is displayed to the user

---

## 🧩 System Design Highlights

* Object-Oriented structure (Person → Student/Admin)
* Separation of logic into multiple modules
* Lightweight data storage using CSV instead of database
* Easy to modify and extend

---

## ⚠️ Limitations

* No graphical user interface (CLI-based system)
* Limited scalability with CSV storage
* No real-time multi-user support
